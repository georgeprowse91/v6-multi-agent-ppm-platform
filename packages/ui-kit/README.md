# UI Kit

UI kit components shared across the platform's front-end applications.
The adjacent design-system directory contains design assets and references that inform
the components, but it does not change the published package outputs.

## Directory structure

| Folder | Description |
|--------|-------------|
| [src/](./src/) | Component source files |
| [design-system/](./design-system/) | Design assets and references that guide component work |

## Key files

| File | Description |
|------|-------------|
| `src/__init__.py` | Module init and public exports |
