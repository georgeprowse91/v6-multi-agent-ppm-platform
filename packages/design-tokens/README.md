# @ppm/design-tokens

Versioned npm package for platform design tokens.

## Exports

- `@ppm/design-tokens` → TypeScript token object
- `@ppm/design-tokens/tokens.css` → CSS custom properties
- `@ppm/design-tokens/tokens.json` → raw JSON tokens

## Consumers

- `packages/ui-kit`
- `apps/web/frontend`
