## Purpose
Plan resource acquisition and allocation aligned to phase-gated delivery, ensuring resources are
available for each stage gate and quality review.

## Audience
Project Manager, Resource Manager, PMO.

## Inputs
- Schedule baseline (`docs/methodology/waterfall/templates/schedule-baseline.xlsx`).
- Shared resource plan (`docs/templates/shared/resource-management-plan.md`).
