# Operations Management

## Overview

Operations management agents handle cross-cutting operational concerns such as change control, release deployment, knowledge management, continuous improvement, stakeholder communications, analytics, data quality, and workflow automation. They keep the platform and its projects running smoothly after initial delivery.

## Directory structure

Each agent directory contains `src/`, `tests/`, a `Dockerfile`, and its own `README.md` unless otherwise noted.
