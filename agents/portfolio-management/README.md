# Portfolio Management

## Overview

Portfolio management agents support the strategic planning layer -- from capturing new demand through building business cases, optimising the portfolio mix, and governing programs of work. They bridge executive strategy with delivery execution.

## Directory structure

Each agent directory contains `src/`, `tests/`, a `Dockerfile`, and its own `README.md`.

| Name | Description | Link |
|------|-------------|------|
| `demand-intake-agent` | Captures, validates, and triages incoming project and initiative requests. | [](/./demand-intake-agent/) |
| `business-case-agent` | Evaluates business cases, cost-benefit analyses, and investment decisions. | [](/./business-case-agent/) |
| `portfolio-optimisation-agent` | Optimises portfolio composition against strategic objectives and constraints. | [](/./portfolio-optimisation-agent/) |
| `program-management-agent` | Coordinates cross-project programs, dependencies, and benefits realisation. | [](/./program-management-agent/) |
