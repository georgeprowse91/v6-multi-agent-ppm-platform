# Portfolio Management

## Overview

Portfolio management agents support the strategic planning layer -- from capturing new demand through building business cases, optimising the portfolio mix, and governing programs of work. They bridge executive strategy with delivery execution.

## Directory structure

Each agent directory contains `src/`, `tests/`, a `Dockerfile`, and its own `README.md`.

| Name | Description | Link |
|------|-------------|------|
| `agent-04-demand-intake` | Captures, validates, and triages incoming project and initiative requests. | [./agent-04-demand-intake/](./agent-04-demand-intake/) |
| `agent-05-business-case-investment` | Evaluates business cases, cost-benefit analyses, and investment decisions. | [./agent-05-business-case-investment/](./agent-05-business-case-investment/) |
| `agent-06-portfolio-strategy-optimisation` | Optimises portfolio composition against strategic objectives and constraints. | [./agent-06-portfolio-strategy-optimisation/](./agent-06-portfolio-strategy-optimisation/) |
| `agent-07-program-management` | Coordinates cross-project programs, dependencies, and benefits realisation. | [./agent-07-program-management/](./agent-07-program-management/) |
