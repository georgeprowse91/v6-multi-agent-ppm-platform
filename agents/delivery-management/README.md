# Delivery Management

## Overview

Delivery management agents cover the full project delivery lifecycle -- from initial scope definition through financial tracking, vendor management, quality assurance, risk mitigation, and regulatory compliance. They provide the operational backbone for executing and governing projects within the portfolio.

## Directory structure

Each agent directory contains `src/`, `tests/`, a `Dockerfile`, and its own `README.md`.

| Name | Description | Link |
|------|-------------|------|
| `scope-definition-agent` | Captures and maintains project scope, objectives, and deliverables. | [](/./scope-definition-agent/) |
| `lifecycle-governance-agent` | Enforces stage-gate governance and lifecycle state transitions. | [](/./lifecycle-governance-agent/) |
| `schedule-planning-agent` | Creates and manages project schedules, milestones, and dependencies. | [](/./schedule-planning-agent/) |
| `resource-management-agent` | Plans and monitors resource allocation and capacity utilisation. | [](/./resource-management-agent/) |
| `financial-management-agent` | Tracks budgets, costs, forecasts, and financial health of projects. | [](/./financial-management-agent/) |
| `vendor-procurement-agent` | Manages vendor relationships, contracts, and procurement processes. | [](/./vendor-procurement-agent/) |
| `quality-management-agent` | Defines quality criteria and monitors compliance with quality standards. | [](/./quality-management-agent/) |
| `risk-management-agent` | Identifies, assesses, and tracks risks and issues across projects. | [](/./risk-management-agent/) |
| `compliance-governance-agent` | Ensures projects meet regulatory and compliance requirements. | [](/./compliance-governance-agent/) |
