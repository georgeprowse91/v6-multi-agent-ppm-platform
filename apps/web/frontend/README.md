# Web Frontend

React frontend for the web console, built with Vite.

## Directory structure

| Folder | Description |
| --- | --- |
| [public/](./public/) | Static public assets |
| [scripts/](./scripts/) | Build and development scripts |
| [src/](./src/) | Frontend source code |

## Key files

| File | Description |
| --- | --- |
| `package.json` | Node.js dependencies and scripts |
| `tsconfig.json` | TypeScript compiler configuration |
| `tsconfig.node.json` | TypeScript configuration for Node.js tooling |
| `vite.config.ts` | Vite bundler configuration |
| `vitest.config.ts` | Vitest test runner configuration |
| `index.html` | HTML entry point |
