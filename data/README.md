# Data

Data assets for the platform, including fixture data, lineage definitions, database migrations, data quality rules, canonical JSON schemas, and seed data.

## Directory structure

| Folder | Description |
|--------|-------------|
| [fixtures/](./fixtures/) | Test fixture data (tax rates, exchange rates) |
| [lineage/](./lineage/) | Lineage data assets |
| [migrations/](./migrations/) | Database migration scripts (Alembic) |
| [quality/](./quality/) | Data quality rules |
| [schemas/](./schemas/) | Canonical JSON schemas for platform entities |
| [seed/](./seed/) | Seed data for initial platform setup |
