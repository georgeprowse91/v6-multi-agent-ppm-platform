# SQLAlchemy Stubs

## Overview

This directory contains a local SQLAlchemy stub/shim package used by the platform. It provides minimal type stubs and module placeholders for SQLAlchemy components referenced elsewhere in the codebase.

## Directory structure

| Folder | Description |
| --- | --- |
| [ext/](./ext/) | SQLAlchemy extension stubs (asyncio.py) |
| [orm/](./orm/) | SQLAlchemy ORM stubs |

## Key files

| File | Description |
| --- | --- |
| [__init__.py](./__init__.py) | Package initializer |
